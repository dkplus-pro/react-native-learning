# React Native Learning

面向有 7 年前端经验的架构师的 React Native 渐进课程仓库。

课程以 Expo + TypeScript 为主线，从零开始，每一课包含：

- 中文课程文档
- 带注释的示例代码
- 可独立运行的 demo
- 独立安装与运行说明

## 快速开始

```bash
npm install
npm run typecheck
npm test
npm run lint
```

运行任意课程 demo：

```bash
cd lessons/lesson-01
npm install
npm start
```
